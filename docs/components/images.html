# Usage

An image element may be used to embed an image on a web page. One should be careful to use images intentionally to avoid overwhelming the user with visual information. They should be surrounded and supported by text. For more detailed information about images on the web, please see the Mozilla web docs. Please also review UAB's guidelines on web images.

# Markup

<img src="LINK TO RAW IMAGE FILE" alt="DESCRIPTION OR PURPOSE OF IMAGE">

If the image is used for decorative purposes only, leave the alt attribute empty but do not remove the alt attribute altogether as it is required for accessibility purposes.

# Captions

An image caption, or more accurately a figure caption, is a set of standard HTML elements used to semantically group a visual piece of information with a caption. Please see the Mozilla web docs for more information on the figure elements.

<figure>
  <img src="Image-URL" alt="Title of image">
  <figcaption>Image caption</figcaption>
</figure>
